# static-d-day
D-Day 를 계산해서 보여줍니다.

## Supported params

### Date
- since: 기준일
- format: 계산 단위 (hour/day/month/year)

### Text
- prefix: 접두
- suffix: 접미

### Style
- align: 텍스트 정렬
- color: 텍스트 색상
- bg-color: 배경 색상
