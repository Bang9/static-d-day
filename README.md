# static-d-day
D-Day 를 계산해서 보여줍니다. <br/>
[Notion](https://www.notion.so/me-f1450b59d11540c6ad64d2e7703668f5) 에 임베드해서 보여주기 위해 만들어졌습니다.

## Supported params

### Date
- since: 기준일
- format: 계산 단위 (hour/day/month/year)

### Text
- prefix: 접두
- suffix: 접미

### Style
- align: 텍스트 정렬
- color: 텍스트 색상
- bg-color: 배경 색상
